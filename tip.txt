### Time complexity depending on the range of N.
1. N <= 500
O(N^3)

2. N <= 2000
O(N^2)

3. N <= 100,000
O(NlogN)

4. N <= 10,000,000
O(N)

### Space complexity
1. int a[1000]
4KB

2. int a[1000000]
4MB

3. int a[2000][2000]
16MB
